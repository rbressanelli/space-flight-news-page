# Aplicativo que consome a API Space Flight News

Link funcional: https://space-flight-news-page.vercel.app/

### Tecnologias utilizadas

- NextJS
- Material (MUI)
- Tanstack (React Query)
- Axios

## Procedimentos para instalação local

- Clonar o repositório
- Instalar dependências ( npm install )
- Rodar o aplicativo ( npm run dev )
- Seguir o link local: http://localhost:3000/
